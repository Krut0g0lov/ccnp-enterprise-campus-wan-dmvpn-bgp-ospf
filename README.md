# CCNP Enterprise Lab: Campus + WAN (OSPF Multi-Area + eBGP + DMVPN/IPsec + HSRP + QoS + NAT + ACL + VRF-lite)

- Campus: VLAN/SVI, Rapid-PVST, EtherChannel (LACP), HSRP
- Routing: OSPF multi-area (area 0/10/20), route summarization, stub area
- WAN/Edge: eBGP к ISP, NAT/PAT, ACL
- Overlay: DMVPN Phase 3 + IPsec (IKEv2), NHRP, spoke-to-spoke
- Segmentation: VRF-lite (GUEST) + special policy routing in WAN
- QoS: LLQ для Voice + shaping на WAN
